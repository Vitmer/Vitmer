## AWS Platform Engineer / DevSecOps Architect

I design and build cloud platforms for AWS and Azure environments with a focus on
multi-account governance, reusable Terraform/OpenTofu modules, Kubernetes
runtime platforms, CI/CD automation, and production operations.

Most client work cannot be published in full because of confidentiality and NDA
constraints. The public repositories here are therefore selected reference
implementations and portfolio projects that show the architecture patterns,
automation style, and engineering practices I use in real platform work.

### What I Build

- AWS multi-account platforms, landing zones, and governance baselines
- Terraform, OpenTofu, and Terragrunt module frameworks
- Hub-and-spoke networking, account isolation, IAM/SCP controls, and audit trails
- AMI/Image Factory automation with EC2 Image Builder and AWS Systems Manager
- Kubernetes platforms, GitOps delivery, CI/CD pipelines, and runtime automation
- Monitoring, logging, security baselines, cost controls, and operational hardening
